Abyssal Aegis is available. Use `/aegis` to invoke the command center for
structured task analysis, planning, auditing, research, and maintenance.

Key modes: `--think` (brainstorm), `--plan` (architecture), `--crit` (adversarial review),
`--guard` (watchdog build), `--research` (deep investigation), `--audit` (post-task check),
`--maintain` (overdue task sweep), `--wrap` (session-end synthesis).

Run `/aegis --help` for the full mode list.
